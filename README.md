# Vaishnava Vibes - Ultimate Chanting Portal

A premium, immersive digital chanting experience designed to help devotees focus on the Hare Krishna Mahamantra. This application combines modern web technologies with spiritual aesthetics to create a serene environment for meditation.

## Features ‚ú®

- **Digital Bead Counter:** Tactile interaction with visual ripple effects and bead tracking.Track your rounds (108 beads/round) and daily goals.
- **Divine Atmospherics:**
  - **3D Particle Vortex:** A mesmerizing, soothing background powered by Three.js.
  - **Procedural Audio:** Organic wood-click sounds and temple bells generated in real-time (no large files).
  - **High-Quality Art:** "Darshan Mode" featuring beautiful representations of the divine.
- **Progress Tracking:** Persistently tracks your total lifetime chants locally. tracking of daily rounds.
- **Immersive Modes:**
  - **Darshan Mode:** Focus specifically on the deity forms.
  - **Drone Music:** (Coming Soon) Background tampura drone for deeper focus.

## Usage üìø

1. Open `index.html` in any modern web browser.
2. Click or tap the central "Bead" button to chant _Hare Krishna Hare Krishna, Krishna Krishna Hare Hare, Hare Rama Hare Rama, Rama Rama Hare Hare_.
3. Use the **Darshan Eye** icon to toggle the visual focus mode.
4. Use the **Sound** icon to toggle bead sounds.
5. Complete 108 beads to finish a round!
